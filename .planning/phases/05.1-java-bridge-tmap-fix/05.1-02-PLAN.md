```

      - region: "{{java}}countries.region"  (cross-table)
      - salary_grade: "{{java}}row1.salary >= 75000 ? \"Senior\" : \"Junior\""  (ternary - THE BUG)
tFileOutputDelimited_1 (output.csv, delimiter=",") -> writes enriched output
```
