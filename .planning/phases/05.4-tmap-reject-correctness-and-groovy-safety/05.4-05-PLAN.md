       ```python
       def _groovy_escape_expression(self, java_expr: str) -> str:
           """Escape Groovy-special characters in a Java expression string before
           embedding into a Groovy script line.

           Tokenizes the expression to identify double-quoted string literal regions
           (respecting Java escape rules: \\\\ and \\" do not end the string).
           Inside string literal regions, escapes characters that Groovy parses
           differently from Java.  Outside string literal regions, passes the text
           through unchanged because Java and Groovy share operator/identifier syntax.

           Currently applied escapes (from 05.4-GROOVY-AUDIT.md):
           - $ inside a string literal -> \\$ (prevents Groovy GString interpolation)

           Args:
               java_expr: Raw Java/Talend expression text (with {{java}} prefix already
                   stripped by _strip_java_marker).

           Returns:
               Expression string safe for embedding into a Groovy source line.
           """
           result: list[str] = []
           in_string = False
           i = 0
           while i < len(java_expr):
               ch = java_expr[i]
               if not in_string:
                   if ch == '"':
                       in_string = True
                       result.append(ch)
                       i += 1
                   else:
                       result.append(ch)
                       i += 1
               else:
                   # Inside a double-quoted string literal.
                   if ch == '\\' and i + 1 < len(java_expr):
                       # Java escape sequence -- pass both chars through unchanged.
                       result.append(ch)
                       result.append(java_expr[i + 1])
                       i += 2
                   elif ch == '"':
                       # End of string literal.
                       in_string = False
                       result.append(ch)
                       i += 1
                   elif ch == '$':
                       # Groovy GString interpolation trigger -- escape it.
                       result.append('\\$')
                       i += 1
                   else:
                       result.append(ch)
                       i += 1
           return "".join(result)
       ```
