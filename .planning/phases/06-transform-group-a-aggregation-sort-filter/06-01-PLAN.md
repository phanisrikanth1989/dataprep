```python
_SUPPORTED_FUNCTIONS = frozenset({
    "count", "min", "max", "avg", "sum", "first", "last",
```

**_validate_config():**
- Validate `operations` is a list
- Each operation dict must have `function` and `input_column` keys
- Each operation's `function` must be in `_SUPPORTED_FUNCTIONS`
- Raise `ConfigurationError` with `[{self.id}]` prefix for any violation

**_process(input_data):**
1. Early return if input_data is None or empty: `return {"main": pd.DataFrame(), "reject": None}`
2. Read config: `groupbys = self.config.get("groupbys", [])`, `operations = self.config.get("operations", [])`, `list_delimiter = self.config.get("list_delimiter", ",")`, `use_financial_precision = self.config.get("use_financial_precision", True)`
3. Build group columns list: `group_input_cols = [g["input_column"] for g in groupbys]`, `group_output_cols = [g["output_column"] for g in groupbys]`
4. If no groupbys (global aggregation): aggregate entire DataFrame
5. If groupbys present: use `df.groupby(group_input_cols)` with single-pass aggregation

**Aggregation logic -- single pass (AGGR-05, D-13):**
- For each operation, build the appropriate aggregation function
- Handle `ignore_null` per operation: when True (default), use `skipna=True`; when False, use `skipna=False` (per D-11)
- Handle `use_financial_precision` (D-12): when True, convert numeric input columns to `Decimal` before aggregation, use pure Python Decimal arithmetic for std/mean/var
- Handle each function:
  - `count`: `"count"` (always works with NamedAgg)
  - `sum`: `lambda x: x.sum(skipna=ignore_null)` or Decimal sum
  - `avg`/`mean`: `lambda x: x.mean(skipna=ignore_null)` or Decimal mean
  - `min`: `lambda x: x.min(skipna=ignore_null)`
  - `max`: `lambda x: x.max(skipna=ignore_null)`
  - `first`: `"first"`
  - `last`: `"last"`
  - `std`: `lambda x: x.std(ddof=1, skipna=ignore_null)` or Decimal std with ddof=1 (D-10)
  - `population_std_dev`: `lambda x: x.std(ddof=0, skipna=ignore_null)` or Decimal std with ddof=0 (D-10)
  - `variance`: `lambda x: x.var(ddof=1, skipna=ignore_null)`
  - `median`: `lambda x: x.median(skipna=ignore_null)`
  - `count_distinct`: `lambda x: x.nunique()`
  - `list`/`list_object`: `lambda x: list_delimiter.join(x.dropna().astype(str))` if ignore_null else `lambda x: list_delimiter.join(x.astype(str))` (D-09)
  - `union`: same as list (log warning that union has no distinct Talend engine behavior)

**Single-pass groupby pattern:**
```python
# Build aggregation dict for groupby
agg_specs = {}
for op in operations:
    output_col = op.get("output_column", op["input_column"])
    input_col = op["input_column"]
    func = op["function"]
    ignore_null = op.get("ignore_null", True)
    agg_specs[output_col] = (input_col, _build_agg_func(func, ignore_null, list_delimiter, use_financial_precision))

if group_input_cols:
    result = df.groupby(group_input_cols, sort=False).agg(**{
        out_col: pd.NamedAgg(column=spec[0], aggfunc=spec[1])
        for out_col, spec in agg_specs.items()
    }).reset_index()
    # Rename groupby columns from input_column to output_column
    rename_map = {g["input_column"]: g["output_column"] for g in groupbys if g["input_column"] != g["output_column"]}
    if rename_map:
        result = result.rename(columns=rename_map)
else:
    # Global aggregation (no groupby)
    result_row = {}
    for out_col, (in_col, agg_func) in agg_specs.items():
        if callable(agg_func):
            result_row[out_col] = agg_func(df[in_col])
        else:
            result_row[out_col] = getattr(df[in_col], agg_func)()
    result = pd.DataFrame([result_row])
```

**Helper method `_build_agg_func`** (static or module-level):
A private function that returns the appropriate callable or string for a given function name, taking ignore_null, list_delimiter, and use_financial_precision as parameters. For Decimal precision mode, convert the series to Decimal objects and compute using pure Python (avoiding pandas numpy delegation that fails on Decimal).

**Decimal precision helpers** (private module-level functions):
- `_decimal_sum(series)`: `sum(Decimal(str(x)) for x in series.dropna())`
- `_decimal_mean(series)`: `_decimal_sum(series) / Decimal(str(series.dropna().count()))`
- `_decimal_std(series, ddof)`: Manual computation using Decimal arithmetic

**Output column ordering:** Ensure result DataFrame has columns in order: group_output_cols first, then operation output_cols in order they appear in operations list.
