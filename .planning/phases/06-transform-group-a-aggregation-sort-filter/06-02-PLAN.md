**Registration:** `@REGISTRY.register("SortRow", "tSortRow")`

**Valid sort types** -- module-level constant:
```python
_VALID_SORT_TYPES = frozenset({"num", "alpha", "date"})
_VALID_ORDERS = frozenset({"asc", "desc"})
```

**_validate_config():**
- `criteria = self.config.get("criteria", [])`
- If not criteria or not isinstance(criteria, list): raise `ConfigurationError(f"[{self.id}] 'criteria' must be a non-empty list")`
- For each criterion in criteria:
  - Must have `"column"` key
  - `sort_type` must be in `_VALID_SORT_TYPES` (default `"alpha"`)
  - `order` must be in `_VALID_ORDERS` (default `"asc"`)

**_process(input_data):**
1. Early return if input_data is None or empty: `return {"main": pd.DataFrame(), "reject": None}`
2. Read config: `criteria = self.config.get("criteria", [])`
3. Log external flag if True: `if self.config.get("external", False): logger.info(f"[{self.id}] external=True noted (not implemented -- using pandas sort_values)")`
4. Filter criteria to only columns present in input_data:
   ```python
   valid_criteria = [c for c in criteria if c["column"] in input_data.columns]
   if not valid_criteria:
       logger.warning(f"[{self.id}] No sort criteria columns found in input data, returning as-is")
       return {"main": input_data.copy(), "reject": None}
   ```
5. Build sort parameters:
   ```python
   columns = [c["column"] for c in valid_criteria]
   ascending = [c.get("order", "asc") == "asc" for c in valid_criteria]
   sort_types = {c["column"]: c.get("sort_type", "alpha") for c in valid_criteria}
   ```
6. Build key function for type-based sorting:
   ```python
   def sort_key(col: pd.Series) -> pd.Series:
       st = sort_types.get(col.name, "alpha")
       if st == "num":
           return pd.to_numeric(col, errors="coerce")
       elif st == "date":
           return pd.to_datetime(col, errors="coerce")
       else:  # "alpha" -- default string sort
           return col
   ```
7. Sort:
   ```python
   result = input_data.sort_values(
       by=columns,
       ascending=ascending,
       key=sort_key,
       ignore_index=True,
       kind="stable",
       na_position="last",
   )
   ```
8. Stats:
   ```python
   rows = len(input_data)
   self._update_stats(rows, rows, 0)
   logger.info(f"[{self.id}] Sorted {rows} rows by {len(valid_criteria)} criteria")
   ```
9. Schema validation:
   ```python
   if self.output_schema:
       result = self.validate_schema(result, self.output_schema)
   ```
10. Return: `return {"main": result, "reject": None}`
