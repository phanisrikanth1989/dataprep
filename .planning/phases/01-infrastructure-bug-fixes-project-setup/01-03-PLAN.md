
3. **Fix ENG-18 -- Skip code fields**: Define as class-level constant:
```python
SKIP_RESOLUTION_KEYS: frozenset[str] = frozenset({'java_code', 'imports', 'python_code'})
```

4. **Fix NEW-02 -- List-of-dict recursion**: Add `_resolve_list(self, items: list) -> list` that recurses into str (resolve_string), dict (resolve_dict), and list (_resolve_list) elements. Call this from resolve_dict when `isinstance(value, list)`.

5. **resolve_dict must NEVER mutate input**: Always create new dict/list. No `config[key] = resolved_value` -- use `resolved = {}; resolved[key] = ...`.

6. **Resolution patterns** (preserve from current implementation):

   - `load_from_file(self, file_path, delimiter='=')` -- load key=value file
   - `set(self, key, value, value_type=None)` -- store with type conversion
   - `get(self, key, default=None)` -- retrieve
   - `resolve_string(self, text)` -- resolve context patterns in a string
   - `resolve_dict(self, config)` -- resolve all patterns in a config dict tree
   - `get_all(self)` -- return copy of all context variables
