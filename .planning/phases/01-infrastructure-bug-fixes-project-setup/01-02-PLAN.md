1. **Fix ENG-02**: `get(self, key: str, default: Any = None) -> Any` -- add the missing `default` parameter. The current code references an undefined `default` variable on line 28.

2. **Class structure:**
```python
class GlobalMap:
    """Talend-compatible globalMap -- key-value store for component statistics and inter-component variables."""
    
    def __init__(self):
        self._store: dict[str, Any] = {}
        self._component_stats: dict[str, dict[str, Any]] = {}
```

3. **Required methods with exact signatures:**
   - `put(self, key: str, value: Any) -> None` -- store value
   - `get(self, key: str, default: Any = None) -> Any` -- retrieve with default (FIXES ENG-02)
   - `remove(self, key: str) -> None` -- delete key (no error if missing)
   - `contains(self, key: str) -> bool` -- check key existence
   - `put_component_stat(self, component_id: str, stat_name: str, value: Any) -> None` -- store stat in both `_component_stats` dict and flat `_store` as `{component_id}_{stat_name}`
   - `get_component_stat(self, component_id: str, stat_name: str, default: Any = 0) -> Any` -- retrieve stat, check `_component_stats` first, fall back to flat key
   - `reset_component(self, component_id: str) -> None` -- NEW: clear all stats for one component (iterate re-execution). Remove from `_component_stats[component_id]` and remove flat keys `{component_id}_*` from `_store`.
   - `get_all(self) -> dict[str, Any]` -- return COPY of `_store` (prevent external mutation)
   - `clear(self) -> None` -- clear both `_store` and `_component_stats`

4. **Logging:** Use `logger = logging.getLogger(__name__)` with `logger.debug()` for put/remove operations. No print() statements (per D-01, ENG-11).

5. **ENG-01 note:** Research debunked ENG-01 (_update_global_map crash). The rewrite inherently avoids this since it's a clean implementation.

6. **Type hints:** Use Python 3.10+ syntax (`dict[str, Any]` not `Dict[str, Any]`).
