   - `__init__(self, enable=True, routines=None, libraries=None, log_level="INFO", lib_dir=None)`
   - Store as `self.lib_dir = lib_dir`
   - Pass to bridge: `self.bridge.start(port=self.port, log_level=self.log_level, lib_dir=self.lib_dir)`

3. **Fail fast on bridge failure (per D-11):**
   Replace the silent fallback pattern:
   ```python
   # REMOVE this pattern:
   except Exception as e:
       logger.error(f"Failed to start Java bridge: {e}")
       logger.warning("Java execution will be disabled...")
       self.enable = False  # <-- silent fallback
   ```
   Replace with:
   ```python
   except Exception as e:
       logger.error(f"Failed to start Java bridge: {e}", exc_info=True)
       self.bridge = None
       self.is_running = False
       raise JavaBridgeError(f"Failed to start Java bridge: {e}") from e
   ```

4. **Add exc_info=True to error logging** for traceback visibility:
   - `logger.error(f"Failed to start Java bridge: {e}", exc_info=True)`
   - `logger.error(f"Error stopping Java bridge: {e}", exc_info=True)`
   - `logger.error(f"Failed to load {routine_class}", exc_info=True)`

5. **Import JavaBridgeError:**
   Add `from src.v1.engine.exceptions import JavaBridgeError` at top

6. **Replace generic RuntimeError with JavaBridgeError:**
   Change `raise RuntimeError(error_msg)` (line 61) to `raise JavaBridgeError(error_msg)`

7. **Update type hints to Python 3.10+ syntax:**
   Change `Optional[List[str]]` to `list[str] | None`
   Change `List[str]` to `list[str]`
   Remove `from typing import Optional, List`

Keep all existing functionality: dynamic port allocation, context manager pattern, library validation, routine loading.
