## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-05 | Tampering | ArrowSerializer.mapSchemaTypeToJava | mitigate | Unknown types log warning and default to String.class -- no crash but no silent incorrect type coercion. Schema validated on Python side first. |
| T-02-06 | Denial of Service | compiledScriptClasses cache | accept | ConcurrentHashMap grows unbounded per job. Single-threaded batch ETL means one job at a time; JVM process killed after job. Not a production concern. |
| T-02-07 | Elevation of Privilege | loadRoutine Class.forName | mitigate | Only loads classes already on classpath (JARs validated by validateLibraries). No dynamic classpath modification at runtime. |
| T-02-08 | Denial of Service | RootAllocator(Long.MAX_VALUE) | accept | Existing architecture. Per-job JVM process isolation limits blast radius. Memory limit is a future optimization, not a correctness issue. |
</threat_model>
