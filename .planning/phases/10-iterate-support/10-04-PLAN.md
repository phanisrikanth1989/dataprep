<objective>
Implement the `tFlowToIterate` engine component
(`src/v1/engine/components/iterate/flow_to_iterate.py`) extending `BaseIterateComponent`.
Iterates each input row via `df.to_dict('records')`. Per row: writes per-column globalMap
keys using either DEFAULT_MAP=true (`<inputFlow>.<col>`) or DEFAULT_MAP=false (custom
map_entries with verbatim user-defined keys). 1-based CURRENT_ITERATION counter set
BEFORE body runs. After all iterations, NB_LINE = total input rows. Registers under
both `FlowToIterate` and `tFlowToIterate` aliases.

Purpose: tFlowToIterate is THE row-to-iterate bridge in Talend production jobs. The .item
sample `Job_tFlowToIterate_0.1.item` reads `globalMap.get("row1.filepath")` per iteration,
confirming the DEFAULT_MAP=true key format `<inputFlow>.<col>` (D-F3).

Output: New `iterate/` engine package, FlowToIterate component, comprehensive unit tests.
</objective>

       @REGISTRY.register("FlowToIterate", "tFlowToIterate")

           Iterates each input row, putting per-row variables into globalMap. Per

    5. Methods (validate -> process -> helpers):

       a. `_validate_config(self) -> None` (STRUCTURAL ONLY):
          - `default_map` must be bool (or coerce-able from "true"/"false" strings)
          - `map_entries` must be a list (default empty); each entry must be a dict
            with `key` and `value` string keys -- only enforced when default_map=False
          - `self.inputs` must be non-empty (this is structural -- engine sets it
            from JSON at engine init, NOT a context-resolved value, so this check is
            allowed in _validate_config per Phase 7.1 Rule 12)
          - Raise ConfigurationError on any violation with `[{self.id}]` prefix

       b. `prepare_iterations(self, input_data: Optional[pd.DataFrame] = None) -> Iterator[FlowToIterateItem]`:
          - If `input_data is None`: raise ConfigurationError(f"[{self.id}] tFlowToIterate requires a non-None input DataFrame")
          - If `input_data.empty`: set `self.total_iterations = 0`, return `iter([])`
          - records = `input_data.to_dict('records')` (D-F2)
          - `self.total_iterations = len(records)`
          - Yield `FlowToIterateItem(row=row, index=idx)` for idx, row in enumerate(records, start=1)
          - Materialize into list-iter via `iter([FlowToIterateItem(...) for ...])` so total_iterations is correct (consumer-friendly bounded iterator)

       c. `set_iteration_globalmap(self, item: FlowToIterateItem) -> None`:
          ```python
          if self.global_map is None:
              return
          default_map = bool(self.config.get("default_map", True))
          if default_map:
              # D-F3: per-column key with input flow prefix
              if not self.inputs:
                  raise ConfigurationError(
                      f"[{self.id}] DEFAULT_MAP=true requires a connected input flow; "
                      f"self.inputs is empty"
                  )
              flow_prefix = self.inputs[0]
              for col, value in item.row.items():
                  # pd.NA -> None for Java bridge compatibility (RESEARCH Risk 10.2)
                  safe_value = None if value is pd.NA else value
                  # Defensive str() coercion for non-string column names (Risk 10.2)
                  self.global_map.put(f"{flow_prefix}.{str(col)}", safe_value)
          else:
              # D-F4: per-entry custom mapping
              map_entries = self.config.get("map_entries", [])
              for entry in map_entries:
                  key = entry.get("key", "")
                  value_col = entry.get("value", "")
                  if value_col not in item.row:
                      raise ConfigurationError(
                          f"[{self.id}] map_entries references column '{value_col}' "
                          f"not in input row (available: {list(item.row.keys())})"
                      )
                  raw_value = item.row[value_col]
                  safe_value = None if raw_value is pd.NA else raw_value
                  self.global_map.put(key, safe_value)
          ```

       d. `finalize(self) -> None`:
          - Set `self.stats["NB_LINE"] = self.total_iterations`
          - Set `self.stats["NB_LINE_OK"] = self.total_iterations`
          - Set `self.stats["NB_LINE_REJECT"] = 0`
          - Logging note: Executor's `_update_global_map()` will publish `{cid}_NB_LINE`
