<objective>
Extract `ENABLE_PARALLEL` and `NUMBER_PARALLEL` from ITERATE-typed connection
elementParameter children in the converter. Plumb a generic `params: dict` on
`TalendConnection` so future connection-level extractions don't require new fields.
Emit an `engine_gap` `needs_review` entry when `ENABLE_PARALLEL=true` warning the
user that Phase 10 runs sequentially.

Purpose: Talend production .item files set `ENABLE_PARALLEL=true` for some iterate
connections. Without this extraction, the engine silently runs sequentially with no
visibility. The `engine_gap` entry surfaces this in audit reports and protects users.

Output: Modified converter + xml_parser + dataclass; comprehensive unit test using both
.item fixtures.
</objective>

    Read the existing implementation (around lines 219-239) to find the exact loop

    Current behavior: returns `flows: list[dict]`.
    New behavior: returns `(flows, needs_review_entries)` tuple.

    ```python
    @staticmethod
    def _parse_flows(connections):
        flows = []
        needs_review_entries = []
        for conn in connections:
            if conn.connector_type not in _FLOW_CONNECTOR_TYPES:
                continue
            if not conn.source or not conn.target:
                continue
            flow = {
                "name": conn.name or conn.source,
                "from": conn.source,
                "to": conn.target,
                "type": conn.connector_type.lower(),
            }
            if conn.connector_type == "ITERATE":
                enable_parallel_raw = conn.params.get("ENABLE_PARALLEL", "false")
                enable_parallel = str(enable_parallel_raw).strip().lower() == "true"
                number_parallel_raw = conn.params.get("NUMBER_PARALLEL", "0") or "0"
                try:
                    number_parallel = int(number_parallel_raw)
                except (TypeError, ValueError):
                    number_parallel = 0
                flow["enable_parallel"] = enable_parallel
                flow["number_parallel"] = number_parallel
                if enable_parallel:
                    needs_review_entries.append({
                        "severity": "engine_gap",
                        "component_id": conn.source,
                        "message": (
                            f"Parallel iteration is configured "
                            f"(NUMBER_PARALLEL={number_parallel}) but Phase 10 engine "
                            f"runs sequentially -- results correct but slower. Defer "
                            f"to Phase 12+ for parallel."
                        ),
                    })
            flows.append(flow)
        return flows, needs_review_entries
    ```
