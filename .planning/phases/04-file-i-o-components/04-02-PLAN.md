**_process(self, input_data=None) -> dict:** This is a sink component (receives input_data DataFrame). Steps:

1. **Read config values** (per D-04, D-05): Read all config keys using Talend defaults:
   - `filepath` (str), `fieldseparator` (str, default ";"), `row_separator` (str, default "\\n"), `encoding` (str, default "ISO-8859-15"), `include_header` (bool, default False -- CRITICAL: Talend defaults False, old engine defaulted True), `append` (bool, default False), `csv_option` (bool, default False), `escape_char` (str, default '"'), `text_enclosure` (str, default '"'), `os_line_separator` (bool, default True), `csvrowseparator` (str, default "LF" -- NOTE: key is `csvrowseparator` without underscores, verified from converter sample JSON), `create_directory` (bool, default True), `split` (bool, default False), `split_every` (str, default "1000"), `delete_empty_file` (bool, default False), `file_exist_exception` (bool, default True), `die_on_error` (bool, default False)

2. **Warn on deferred features** (per D-21): Check flags `compress`, `usestream`, `row_mode`, `flushonrow`. If any is truthy, log `logger.warning(f"[{self.id}] {description} ('{flag}') is not yet implemented. Config flag will be ignored.")`.

3. **Resolve filepath:** Convert via `Path(filepath)`.

4. **Create directory** (per D-16): If `create_directory=True` and parent directory does not exist, create it: `resolved_path.parent.mkdir(parents=True, exist_ok=True)`.

5. **FILE_EXIST_EXCEPTION check** (per D-17, FOLD-05): If `file_exist_exception=True` and `append=False` and file exists, raise `FileOperationError(f"[{self.id}] File already exists: '{filepath}'. Set file_exist_exception=false or append=true to allow writing.")`.

6. **Handle empty input** (addresses review concern: explicitly handle header-only file for empty input):
   If `input_data is None` or `input_data.empty`:
   - If `delete_empty_file=True`, do not create file (or delete if exists). Return `{"main": None, "reject": None}`.
   - If `include_header=True` and `self.output_schema` or (input_data is not None and has columns): write a header-only file containing just the column names row with the configured delimiter and encoding. This matches standard Talend behavior where an empty flow with include_header=True produces a file with only the header line.
   - If `include_header=False`: write an empty file (0 bytes).
   - Return `{"main": input_data, "reject": None}`.

7. **Determine effective line separator:**
   - If `os_line_separator=True`: use `os.linesep`
   - Elif `csv_option=True`: use `_resolve_csv_row_separator(csvrowseparator)` where mapping is `{"LF": "\n", "CR": "\r", "CRLF": "\r\n"}`. Also handle raw values like `"\\n"` via `_unescape_separator()`.
   - Else: use `_unescape_separator(row_separator)`

8. **Unescape field separator:** Use `_unescape_separator(fieldseparator)`.

9. **SPLIT mode** (per D-18, FOLD-04): If `split=True`:
   - Parse `split_every` to int (default 1000).
   - Split DataFrame into chunks of `split_every` rows.
   - For each chunk at index `i`, compute filename using `_split_filename(filepath, i)` which produces `{stem}{i}{suffix}` (e.g., "output0.csv", "output1.csv").
   - Write each chunk to its split file.

10. **Write file(s):** For each output file:
    - If `csv_option=True`: Use `csv.writer` with `delimiter=field_separator`, `quotechar=text_enclosure`, `escapechar=escape_char if escape_char != text_enclosure else None`, `doublequote=(escape_char == text_enclosure)`, `quoting=csv.QUOTE_MINIMAL`. Write header row if `include_header=True`. Write data rows. Use effective line separator.
    - If `csv_option=False`: Use `df.to_csv()` with `sep=field_separator`, `header=include_header`, `index=False`, `encoding=encoding`, `quoting=csv.QUOTE_NONE`, `lineterminator=effective_line_separator`, `mode='a' if append else 'w'`.

11. **Set globalMap variables** (per D-20, FOLD-06): Set `{self.id}_FILE_NAME` = str(resolved filepath). Also set `{self.id}_NB_LINE` = total rows written (for explicit tracking beyond BaseComponent auto-stats). Guard with `if self.global_map:`.

12. **Return result:** `return {"main": input_data, "reject": None}`. Output components pass through input_data as main.
