    ```python
    class RoutineNamespace:
        """Namespace object that allows routines.RoutineName.method() access pattern."""

        def __init__(self, routines: Dict[str, Any]):
            self._routines = routines

        def __getattr__(self, name: str) -> Any:
            if name.startswith('_'):
                raise AttributeError(name)
            routine = self._routines.get(name)
            if routine is None:
                raise AttributeError(f"No routine named '{name}'. Available: {list(self._routines.keys())}")
            return routine

        def __repr__(self) -> str:
            return f"RoutineNamespace({list(self._routines.keys())})"
    ```

        if not os.path.exists(routines_dir):

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-09-05 | Elevation of Privilege | bridge.py start() | mitigate | Validate JAR paths resolve to real files via os.path.isfile(). Log warnings for missing JARs. Do NOT allow arbitrary command injection via JAR paths -- paths go into -cp argument which is a classpath, not executed directly. |

| T-09-07 | Elevation of Privilege | python_routine_manager.py | mitigate | Only load .py files from the configured routines_dir directory. Skip files starting with underscore. | T-09-08 | Tampering | python_routine_manager.py | accept | Routine code is trusted (deployed by administrators alongside the engine). No sandboxing needed -- routines run with full Python permissions, same as Talend routine.jar behavior. |
| T-09-09 | Information Disclosure | java_bridge_manager.py | mitigate | Log routine class names at INFO level but do not log JAR file contents or routine source code. |
