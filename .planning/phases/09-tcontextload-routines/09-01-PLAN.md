    **_process(self, input_data):**
    Three-phase model per research Pattern 1:

    Phase A -- Setup:
    ```python
    assigned_keys: set[str] = set()
    new_keys: set[str] = set()
    existing_context_keys = set(self.context_manager.context.keys()) if self.context_manager else set()
    ```

    Phase B -- Row Processing (per D-03, D-04):
    - If `input_data is None or input_data.empty`, skip to Phase C.
    - Validate `"key"` and `"value"` columns exist in input_data. If missing, raise `DataValidationError(f"[{self.id}] Input DataFrame must have 'key' and 'value' columns, got: {list(input_data.columns)}")`.
    - Extract data vectorized: `keys = input_data["key"].astype(str).str.strip()`, `values = input_data["value"]`, check `has_type = "type" in input_data.columns`.
    - Iterate over rows (iterate for ContextManager.set, but extraction is vectorized):
      - `key = keys.iloc[i]`; skip if empty string.
      - `val = values.iloc[i]`; convert to None if `pd.isna(val)`.
      - Determine type: if `has_type` and `not pd.isna(types.iloc[i])`, use `str(types.iloc[i])`. Else if `self.context_manager` and `self.context_manager.get_type(key)`, use that. Else `"id_String"` (per D-09).
      - Categorize: if `key in existing_context_keys` then `assigned_keys.add(key)` else `new_keys.add(key)`.
      - Unconditionally assign: `self.context_manager.set(key, val, value_type)` (per research: ALL variables are loaded, policies only control messages).
      - If `print_operations` is True: `logger.info("[%s] Context loaded: %s = %s (type: %s)", self.id, key, val, value_type)`.

    Phase C -- Post-Processing Validation:
    - Compute `unloaded_keys = existing_context_keys - assigned_keys - new_keys`.
    - Call `self._emit_validation_messages(new_keys, unloaded_keys)`.
    - `loaded_count = len(assigned_keys) + len(new_keys)`.

    Set globalMap variables (per research Example 3):
    ```python
    if self.global_map:
        self.global_map.put(f"{self.id}_NB_LINE", loaded_count)
        self.global_map.put(f"{self.id}_NB_CONTEXT_LOADED", loaded_count)  # backward compat
        new_keys_str = ",".join(sorted(new_keys)) if new_keys else ""
        unloaded_keys_str = ",".join(sorted(unloaded_keys)) if unloaded_keys else ""
        self.global_map.put(f"{self.id}_KEY_NOT_INCONTEXT", new_keys_str)
        self.global_map.put(f"{self.id}_KEY_NOT_LOADED", unloaded_keys_str)
    ```

    Log summary and return:
    ```python
    logger.info("[%s] Loaded %d context variables (%d new, %d updated, %d unloaded)",
                self.id, loaded_count, len(new_keys), len(assigned_keys), len(unloaded_keys))
    return {"main": pd.DataFrame()}
    ```

    **_emit_validation_messages(self, new_keys, unloaded_keys):**
    - Read config: `load_new_policy = self.config.get("load_new_variable", "WARNING")`, `not_load_old_policy = self.config.get("not_load_old_variable", "WARNING")`.
    - For each key in sorted `new_keys`: build message `f"New context variable '{key}' not in original job context"`. Call `self._emit_message(message, load_new_policy)`.
    - For each key in sorted `unloaded_keys`: build message `f"Context variable '{key}' not loaded from incoming flow"`. Call `self._emit_message(message, not_load_old_policy)`.

    **_emit_message(self, message, level):**
    - Map policy to log level: `"ERROR"` -> `logger.error`, `"WARNING"` -> `logger.warning`, `"INFO"` -> `logger.info` (per D-05/D-06). `"NO_WARNING"` -> return immediately (no message per research open question 1).
    - Check DISABLE flags (per D-07): `disable_error = self.config.get("disable_error", False)`, `disable_warnings = self.config.get("disable_warnings", True)`, `disable_info = self.config.get("disable_info", True)`.
    - If level is `"ERROR"` and `disable_error` is True: return (suppressed).
    - If level is `"WARNING"` and `disable_warnings` is True: return (suppressed).
    - If level is `"INFO"` and `disable_info` is True: return (suppressed).
    - Emit the message at the appropriate log level with `[{self.id}]` prefix.
    - Per D-08: if level is `"ERROR"` and message was emitted (not suppressed) and `self.config.get("die_on_error", False)` is True, raise `ComponentExecutionError(self.id, message)`.
