Insert this fallback block (before the `for idx in df[ok_mask].index:` per-row loop):

```python
            # Talend MANUALTABLE default: when no output schema provides a target type,
            # attempt pd.to_numeric() for in-place casts (input_column == output_column).
            # This matches tConvertType's default MANUALTABLE behavior (per D-B3).
            # errors="coerce" -> non-numeric values become NaN (no rows rejected).
            if target_dtype == "object" and in_col == out_col:
                inferred = pd.to_numeric(df.loc[ok_mask, in_col], errors="coerce")
                if inferred.notna().any():
                    df.loc[ok_mask, in_col] = inferred
                continue  # skip per-row loop; whole-column operation done
```

Important constraints:
- Only applies when `target_dtype == "object"` (no explicit type declared) AND `in_col == out_col` (in-place cast).
- Does NOT affect cross-column casts (`in_col != out_col`) or casts with an explicit target type.
- Does NOT produce reject rows -- this is best-effort numeric inference matching AUTOCAST semantics.
- The `continue` skips the per-row loop for this column only.
